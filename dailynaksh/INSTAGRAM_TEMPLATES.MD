# VedicDose Instagram Reel Templates

Complete Canva specifications for creating VedicDose video content.

## General Specs
- **Size:** 1080x1920 (9:16 vertical)
- **Safe Zone:** 150px padding top/bottom
- **Colors:** Navy #0a0a1f, Gold #d4af37
- **Fonts:** Montserrat or Poppins
- **Watermark:** @vedic.dose (bottom right, 60% opacity)
- **Duration:** 15-30 seconds
- **Export:** MP4, 1080x1920, 30fps

## Template 1: Rasi Profile (30 seconds)

### Purpose
Zodiac sign personality breakdown

### Design Layout
**Background:**
- Solid navy #0a0a1f
- Add constellation overlay (40% opacity gold)

**Symbol (Left Side):**
- Search Canva: "zodiac [sign] constellation"
- Size: 400x400px
- Position: Left 1/3, centered vertically
- Color: Gold #d4af37, 60% opacity

**Text (Right Side):**
- **Title:** Rasi name (e.g., "TAURUS")
  - Font: Montserrat Bold, 80px
  - Color: Gold #d4af37
  - Position: Top right
  
- **3 Traits:** Bullet points
  - Font: Montserrat Regular, 48px
  - Color: White
  - Line spacing: 1.5
  - Each trait on separate line
  
- **Footer:** "@vedic.dose"
  - Font: Montserrat, 36px
  - Color: Gold #d4af37
  - Position: Bottom right

### Animation
- Fade in title (0-1 sec)
- Reveal traits one by one (1-4 sec each)
- Hold full screen (5 sec)
- Fade to CTA (last 2 sec)

### Example Content
```
TAURUS
- Stable & Patient
- Creative Soul  
- Venus Ruled
@vedic.dose
```
